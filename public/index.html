API REST - Contactos

Esta API se puede emplear para realizar prácticas de AJAX. La API utiliza los verbos GET, POST, PUT y DELETE

Caracteristicas

  • Crea un contacto: petición de tipo POST, necesita la url y los datos del contacto a ser creado, devuelve el objeto creado en caso de suceso de lo contrario devuelve un error.
    Ejemplo: https://taller3-api-contacto.herokuapp.com/contacto [,datos]
  • Recupera todos los contactos guardados: petición de tipo GET, necesita la url, devuelve un array de objetos en caso de suceso de lo contrario devuelve un error.
    Ejemplo: https://taller3-api-contacto.herokuapp.com/contactos
  • Recupera un contacto por su id: petición de tipo GET, necesita la url y el id del contacto a ser recuperado, devuelve un objeto en caso de suceso de lo contrario devuelve un error.
    Ejemplo: https://taller3-api-contacto.herokuapp.com/contacto/{id}
  • Actualiza los datos de un contacto: petición de tipo PUT, necesita la url y el id del contacto a ser actualizado, devuelve un objeto en caso de suceso de lo contrario devuelve un error.
    Ejemplo: https://taller3-api-contacto.herokuapp.com/contacto/{id}[, datos]
  • Elimina un contacto: petición de tipo DELETE, necesita la url y el id del contacto a ser eliminado, devuelve un objeto en caso de suceso de lo contrario devuelve un error.
    Ejemplo: https://taller3-api-contacto.herokuapp.com/contacto/{id}
  • Filtra contactos: petición de tipo GET, necesita la url y el término por el cual filtrar, devuelve un array de objetos en caso de suceso de lo contrario devuelve un error.
    Ejemplo: https://taller3-api-contacto.herokuapp.com/contactos/buscar/{termino}
